# soundbox/lick samples

This subdirectory is derived from Michael L Heuer's `Lick/samples` found
[here](https://github.com/heuermh).  As such the samples are subject 
to the license described there and included [here](COPYING).

Our modifications to simplify internet access via github.io:

* convert .wav to .ogg 
* provide meta-data via `instruments.json`.
